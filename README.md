# Inkflow Tattoo Studio App

A React + Tailwind starter app for tattoo artists who want one place to manage:

- clients
- appointments
- intake forms
- reminders
- notes
- message history

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React icons

## Run it locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## What is included

- dark studio dashboard
- searchable client list
- client detail panel
- appointments panel
- messages panel
- form builder preview area
